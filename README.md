# Turing Machine Simulator

This is a simulator demonstrating how Turing Machine works. The application is made using Unity. This project is made for Language and Automata Theory's assignment

## Supported Turing Machine type: 
- Single-Tape
- Multi-Track (2 tracks)
- Multi-Tape (3 tapes)

## Operations 
- Aritmethic Operations:
  - Addition
  - Substraction
  - Multiplication
  - Division
  - Power
  - Factorial
  - Binary Logarithm
- Temperature Conversion
  - Celcius to Fahrenheit
  - Fahrenheit to Celcius
  - Celcius to Kelvin
  - Kelvin to Celcius
  - Fahrenheit to Kelvin
  - Kelvin to Fahrenheit
